# Blankube

简单的魔方模拟器，[示例](https://edgesider.github.io/blankube/dist/index.html)。

计划的功能有：

- 支持高阶魔方
- 解析执行公式
- 键盘操作
- 手柄操作
- 自动打乱
- 还原计时
- CFOP自动还原

另外，在其中尝试实现了之前的一些想法，比如用协程的思路做响应式，目前看起来效果不错。
